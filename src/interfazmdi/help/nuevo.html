Nuevo

Esta Opcion permite al usuario poder generar un nuevo archivo en blanco dentro de nuestra aplicacion.
Esto lo hacemos mendiante:

  • Un internal JFrame
  • Un DesktopPane
  • Un ciclo For

Una vez que el usuario le da Clic a la opcion Nuevo El Programa convoca a la funcion CrearNota() El cual contiene el siguiente Codigo

Como podemos Observar comienza mandando a llamar medidasPantella, el cual el metodo evalua si el contador es menor a 6 divide la pantalla entre 6 para que se acomode en la primera fila del frame en caso contrario el frame acomoda la ventana hija en una segunda fila posicionandolo al principio.Este metodo utiliza una clase externa la cual se llama Ventanahija la cual genera los internal JFrame. El codigo es el siguiente 

public class VentanaHija extends JInternalFrame{
    private final JScrollPane jSPnotas;
    private final JTextArea jTAnotas;
    private final JPanel JPNotas;
    
    public VentanaHija() {
        jTAnotas = new JTextArea(12,25);
        jSPnotas = new JScrollPane(jTAnotas);
        JPNotas = new JPanel();
        JPNotas.setLayout(new GridLayout(1, 1));
        
	}
    
    public void crearFrame(int i){
        
        JPNotas.add(jSPnotas);
        add(JPNotas);
        setTitle("Ventana "+(i+1));
        visualizar();
    }
    
    public void crearFrame(String path, String contenido){
        jTAnotas.insert(contenido, 0);
        JPNotas.add(jSPnotas);
        add(JPNotas);
        setTitle(path);
        visualizar();
        
    }
    
    public void visualizar(){
        pack();
        setVisible(true);
        setLocation(10, 10);
        setResizable(true);
        setMaximizable(true);
        setClosable(true);
        setIconifiable(true);  
        
    }
    
    public String obtenerTexto(){
        String contenido="";
        contenido=jTAnotas.getText();
        return contenido;
        
    }
    
}


Como podemos ver VentanaHija hereda de JInternalFrame Esto hace que se cree un Frame dentro de un JFrame creado anteriormente como si fuera un JPanel. Dentro de el JFrame se instanciaron un JTextArea del Tamaño del JInternalFrame para asi pudiera verse como un procesador de Textos ademas de que se agrego un JScrollbar el cual aparece cuando el texto es mayor al JInternalFrame no se salga.La Ventana tiene un acomodo tipo GridLayout.
