Mosaico

Esta Opcion Permite al Usuario ver los JFrames generados en Forma de Mosaico, es decir en cuadros alineados uno tras de otro.Cuando el usuario le da Clic a la Opcion Mosaico.se Manda a llamar al metodo acomodoMosaico() la estructura de el metodo es la siguiente: 

    public void acomodoMosaico(){
        int xH;
        int yH;
        for (int i = 0; i < hija.length; i++) {
            medidasPantalla(i+1);
            yH=hija[i].getHeight()/2;
            xH=hija[i].getWidth()/2;
            hija[i].setSize(new Dimension(250, 200));
            hija[i].setLocation(x-xH, y-yH);
        }
        
    }
