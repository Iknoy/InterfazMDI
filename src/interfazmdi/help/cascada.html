Cascada

Al Darle Clic a esta Opcion el Usuario puede modificar el acomodo de los JInternalFrame se genera un oyente que hace un llamado al metodo acomodoCascada().
La Esctructura de el Metodo es el Siguiente: 

    public void acomodoCascada(){
        int xC,xH;
        int yC,yH;
        
        xC=principal.getWidth()/15;
        yC=principal.getHeight()/20;
        
        for (int i = 0; i < hija.length; i++) {
            
            yH=hija[i].getHeight()/2;
            xH=hija[i].getWidth()/2;
            hija[i].setSize(new Dimension(250, 200));
            hija[i].setLocation((xC*(i+3))-xH, (yC*(i+3))-yH);
            hija[i].moveToFront();
        }
        
    }
Como vemos en el Codigo apartir de el ultimo JInternalFrame se hace un acomodo por medio del metodo setLocation() mediante una formula que lo hace para que se visualice las ventanas en cascada
