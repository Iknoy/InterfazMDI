Guardar

La Opcion guardar permite al usuario salvar los cambios que tenga dentro del JInternalFrame para asi poder guardarlo dentro de un archivo en alguna Ubicacion.
El usuario al darle Clic a la Opcion Guardar la aplicacion manda a llamar el metodo guardarNota() el cual esta estructurado de la siguiente forma: 

   public void guardarNota(){

        for (int i = 0; i < hija.length; i++) {
            if (hija[i].isSelected()){
                try{
                    JFileChooser file=new JFileChooser();
                    file.showSaveDialog(principal);
                    File guarda =file.getSelectedFile();
                    if(guarda !=null){ /*guardamos el archivo y le damos el formato directamente, si queremos que se guarde en formato doc lo definimos como .doc*/
                        try (FileWriter save = new FileWriter(guarda+".txt")) {
                            save.write(hija[i].obtenerTexto());
                        }
                        JOptionPane.showMessageDialog(null,"El archivo se a guardado Exitosamente", "Guardado",JOptionPane.INFORMATION_MESSAGE);
                    }
                }
                catch(IOException ex){
                    JOptionPane.showMessageDialog(null, "Su archivo no se ha guardado", "Advertencia",JOptionPane.WARNING_MESSAGE);
                }
                break;
            }
        }
    }
Como podemos observar todo el comportamiento del metodo esta por medio de un for para asi saber en que ventana es la que se guardara el archivo, se instancia una variable de tipo JFileChooser despues con un showSaveDialog se agrega dentro de la ventana principal para mostrar un dialogo de guardar archivo.Dentro del try se instancia el metodo Filewriter con el parametro de el archivo seleccionado.
